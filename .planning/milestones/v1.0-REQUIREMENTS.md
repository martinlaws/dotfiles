## Out of Scope

Explicitly excluded. Reasoning validated during v1.0 development.

| Feature | Reason |
|---------|--------|
| Cross-platform support (Windows/Linux) | Mac-only simplifies implementation significantly |
| GUI interface | CLI is the right tool for developer automation |
| Cloud sync of settings | Local dotfiles repo is simpler and more reliable |
| Automatic updates | User should control when setup runs |
| Custom package managers | Homebrew is the macOS standard |
| Docker/VM for testing | Manual testing on Mac Studio sufficient |
| Configuration templating | *.local pattern handles machine differences |
| Ansible/Chef/Puppet | Overkill for single-machine dotfiles |
